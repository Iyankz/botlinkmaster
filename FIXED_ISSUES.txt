### Issue #1: Color Codes Tidak Ter-render di Output

**Masalah:**
```
\033[1;33mnano .env\033[0m
\033[1;33msu - lab\033[0m
```

Output menampilkan ANSI escape codes mentah (`\033[...`) daripada warna.

**Penyebab:**
- Script `install.sh` menggunakan `echo` tanpa flag `-e`
- Flag `-e` diperlukan untuk memproses escape sequences

**Solusi:**
1. âœ… **install.sh sudah diperbaiki**
   - Semua `echo` dengan warna sekarang menggunakan `echo -e`
   - Output sekarang akan menampilkan warna dengan benar

2. âœ… **install-simple.sh dibuat** (backup)
   - Versi alternatif TANPA warna
   - Gunakan ini jika masih ada masalah dengan colors
   - Output plain text, mudah dibaca
